Explanatory text:

To write text in HTML code, there are many different tags to use.
The tag <p> is used to tell the browser that the entered text should be a paragraph.
The tag <h1> is used for headings.
One example of this is:

How it looks in HTML:

<p> “This is a paragraph.”</p>
<h1> “This is a heading.”</h1>
