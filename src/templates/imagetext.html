Explanatory text:

To add a image to a website in HTML, the tag <img> is used.
In order to select which image to show, the tag is combined with “src”, which is a HTML word for “search”.
The tag <p> is used to tell the browser that the entered text should be a paragraph.
The tag <h1> is used for headings.
