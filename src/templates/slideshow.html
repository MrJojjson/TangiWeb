Explanatory text:

To show a simple slideshow in HTML code, the tag <div> can be used which holds the different images. To show the image the <img> tag is placed within the <div> tag. In order to select which image to show, the <img> tag is combined with “src”, which is a HTML word for “search”. Example below with two images:

How it looks in HTML:

<div class="slideshow-container">
  <div class="mySlides fade">
     <div class="numbertext">1 / 2</div>
    <img src="img/Bird.png">
  </div>
  <div class="mySlides fade">
     <div class="numbertext">2 / 2</div>
    <img src="img/Sloth.png">
  </div>
</div>
